# STAATS Python

**Professional Survey Data Processing System for aplusa Healthcare Market Research**

Modern Python replacement for Excel VBA STAATS macros - 100x faster, unlimited scale, cloud-ready.

---

## Quick Start

### Web Application (Recommended)

```bash
# Install dependencies
pip install -r requirements.txt

# Run web app
streamlit run app.py
```

Access at: http://localhost:8501

### Command Line

```bash
# Process survey data
python staats_cli.py process data.csv config.json -o results.xlsx
```

### Docker Deployment

```bash
docker-compose up -d
```

## Features

- Data import (CSV, Excel)
- Automatic datamap configuration
- 7 recode types
- Filter engine
- Cross-tabulation with significance testing
- Professional Excel export

- **Unlimited rows** (no crashes)
